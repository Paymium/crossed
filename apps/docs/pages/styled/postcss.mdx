# PostCSS plugin

## Installation

Install `@crossed/styled` in your project's dependencies and `modify-selectors` in the development dependencies.

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tabs.Tab>

    ```sh
    pnpm i @crossed/styled
    pnpm i -D modify-selectors
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    ```sh
    npm i @crossed/styled
    npm i -D modify-selectors
    ```

  </Tabs.Tab>
  <Tabs.Tab>
  
    ```sh
    yarn add @crossed/styled
    yarn add -D modify-selectors
    ```
  
  </Tabs.Tab>
</Tabs>

Add postcss plugin

```javascript filename="postcss.config.js"
const {
  default: crossedPlugin,
} = require('@crossed/styled/lib/commonjs/postcss');

/** @type {import('postcss-load-config').Config} */
const config = {
  plugins: {
    // ...,
    ...crossedPlugin()
    // ...,
};

export default config;
```

## Specificity
`@crossed/styled` does not directly apply className props to the DOM; instead, it transforms them into data-class-name attributes.
```typescript
// In
<MyComponent className="foo" />

// Out
<div data-class-name="foo" />
```
So you should use `@crossed/babel-plugin` for transform className css rules
```css
// In
.foo {
  // ...
}

// Out
.foo, [data-class-name~="foo"] {
  // ...
}
```
