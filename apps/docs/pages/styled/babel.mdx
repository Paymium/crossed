# Babel

## Installation

Install `@crossed/babel-plugin`

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tabs.Tab>

    ```sh
    pnpm i @crossed/babel-plugin
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    ```sh
    npm i @crossed/babel-plugin
    ```

  </Tabs.Tab>
  <Tabs.Tab>
  
    ```sh
    yarn add @crossed/babel-plugin
    ```
  
  </Tabs.Tab>
</Tabs>

Add in your babel config file

```typescript
module.exports = function (api) {
  return {
    // ...
    plugins: [
      // ...
      ['@crossed/babel-plugin', { tailwindPath: './tailwind.config.js' }],
    ],
  };
};
```

## Specificity

### Principe

`@crossed/babel-plugin` transform all `className` to `style` props if found in these patterns:

- JSX attribute

```typescript
// In
<MyComponent className="..." />

// Out
<MyComponent style={{/* ... */}} />
```

- In `styled` function

```typescript
// In
styled(Pressable, {
  className: ['...'],
});

// Out
styled(Pressable, {
  props: {
    style: {
      /* ... */
    },
  },
});
```

### Usage

Because of these specificities certain things are not possible to do

```typescript
const config = { className: ['...'] }; // ==> Not transpiled by package
styled(Pressable, config);
```
