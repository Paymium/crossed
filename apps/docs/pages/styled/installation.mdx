# Installation

Before using @crossed/style, it is essential to have Tailwind CSS already installed.. If not, see how [install tailwind](https://tailwindcss.com/docs/installation)

### Install dependencies
<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tabs.Tab>

    ```sh
    pnpm i @crossed/styled @crossed/next-adapter react-native-web
    pnpm i -D modify-selectors
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    ```sh
    npm i @crossed/styled @crossed/next-adapter react-native-web
    npm i -D modify-selectors
    ```

  </Tabs.Tab>
  <Tabs.Tab>
  
    ```sh
    yarn add @crossed/styled @crossed/next-adapter react-native-web
    yarn add -D modify-selectors
    ```
  
  </Tabs.Tab>
</Tabs>

### Add postcss plugin

```javascript filename="postcss.config.js"
const {
  default: crossedPlugin,
} = require('@crossed/styled/lib/commonjs/postcss');

/** @type {import('postcss-load-config').Config} */
const config = {
  plugins: {
    // ...,
    ...crossedPlugin()
    // ...,
};

export default config;
```

See [documentation](/styled/postcss)

### Add Nextjs plugin

```javascript filename="next.config.js"
const { withCrossed } = require('@crossed/next-adapter');

/** @type {import('next').NextConfig} */
const nextConfig = withCrossed({
  /** your next config */
});

module.exports = nextConfig;
```
