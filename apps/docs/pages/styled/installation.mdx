# Installation

Before using @crossed/style, it is essential to have Tailwind CSS already installed.. If not, see how [install tailwind](https://tailwindcss.com/docs/installation)

### Add Nextjs plugin

```javascript filename="next.config.js"
const { withCrossed } = require('@crossed/next-adapter');

/** @type {import('next').NextConfig} */
const nextConfig = withCrossed({
  /** your next config */
});

module.exports = nextConfig;
```

### Add babel plugin

```typescript
const path = require('path');

module.exports = function (api) {
  return {
    // ...
    plugins: [
      // ...
      [
        '@crossed/babel-plugin',
        { tailwindPath: path.resolve(__dirname, './tailwind.config.js') },
      ],
    ],
  };
};
```

See [documentation](/styled/babel)

### Tailwind CSS IntelliSense plugin configuration

```json filename=./.vscode/settings.json
"tailwindCSS.experimental.classRegex": [
  ["[`'\"`]?className[`'\"`]?: ?([^;]*)[;]", "[`'\"`]([^'\"`,;]*)[`'\"`]"]
],
```
If you have monorepo, you can define global tailwind config file
```json filename=./.vscode/settings.json
"tailwindCSS.experimental.configFile": "./packages/design_system/tailwind.config.js",
```
