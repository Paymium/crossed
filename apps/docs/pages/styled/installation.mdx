# Installation

Based you already have tailwind initilized. If not, see how [install tailwind](https://tailwindcss.com/docs/installation)

<Steps>
### Install dependencies
<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tabs.Tab>

    ```sh
    pnpm i @crossed/styled
    pnpm i -D modify-selectors
    ```

  </Tabs.Tab>
  <Tabs.Tab>

    ```sh
    npm i @crossed/styled
    npm i -D modify-selectors
    ```

  </Tabs.Tab>
  <Tabs.Tab>
  
    ```sh
    yarn add @crossed/styled
    yarn add -D modify-selectors
    ```
  
  </Tabs.Tab>
</Tabs>

### Add postcss plugin

```javascript filename="postcss.config.js"
const {
  default: crossedPlugin,
} = require('@crossed/styled/lib/commonjs/postcss');

/** @type {import('postcss-load-config').Config} */
const config = {
  plugins: {
    // ...,
    ...crossedPlugin()
    // ...,
};

export default config;
```
