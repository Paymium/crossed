# Base

## `className` and `props`
When you use `styled` you define base `className` and base `props` of your component.
You have two parameters which are `Base` params, `className` (array of tailwind className) and `props` (props of you base component)

`props` params can set default props of your component

- `as` is polymorphic props (optional)

`className` params can set tailwind class name

## State

State of component is manage by javascript event, onClick (web), onPress (Native), ...  
### State list
- `:hover`
- `:active`
- `:disabled`
- `:focus`

### Order priority
```
focus: 1
hover: 2
active: 3
disabled: 4
```
### Usage
In state keys, you define `className` and `props`  
You can manage state from parent with `states` props added to component styled
