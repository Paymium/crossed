    The `Root` component initializes the Modal provider

    The `Portal` component is use if you want use react portal.


    ### Trigger
    The `Trigger` component trigger open modal.

    ### Title
    Title of modal

    ### Overlay
    Overvaly of modal

    ### Content
    Content of modal

    ### Description

    ## useModalContext

    Retrieve the modal context.

    ### Syntax

    ```typescript
    function useModalContext(): ContextModal {
      // Function body
    }
    ```

    ### Parameters

    None.

    ### Return Value

    An object of type `ContextModal`
