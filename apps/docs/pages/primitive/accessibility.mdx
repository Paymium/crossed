# Accessibility

By default, primitive usage ask required accessibility props.

<Box className="h-[290px]">
  <Image
    src="/accessibilityError.png"
    alt="accessibilityError"
    fill
  />
</Box>

For example, when you create button, `aria-label` should be set ([source](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/button_role#basic_buttons)).  
For disable this behavior, you should add this code

```typescript
interface PrimitiveConfigExtend {
  requireAccessibility: false;
}

declare module '@crossed/primitive' {
  interface PrimitiveCustomConfig extends PrimitiveConfigExtend {}
}
```
