    ## useButtonContext

    Retrieve the button context.

    ### Syntax

    ```typescript
    function useButtonContext(): ContextButton {
      // Function body
    }
    ```

    ### Parameters

    None.

    ### Return Value

    An object of type `ContextButton` with an `id` property of type `string`. The `id` is automatically generated.

    ### Example

    ```typescript
    // Example of using the function
    const result = useButtonContext();
    console.log(result);
    ```

    ### Remarks

    None.

    ## useButtonGroupContext
    Retrieve the context of the button group.

    ### Syntax

    ```typescript
    function useButtonGroupContext(): ButtonGroupContext {
      // Function body
    }
    ```

    ### Parameters

    None.

    ### Return Value

    An object of type `ButtonGroupContext` with the following properties:
    - `orientation`: Type `'horizontal' | 'vertical' | undefined`, default is `'horizontal'`.
    - `grouped`: Optional, type `boolean`. Indicates whether the button is wrapped by a group component (`Group`).

    ### Example

    ```typescript
    // Example of using the function
    const result = useButtonGroupContext();
    console.log(result);
    ```

    ### Remarks

    None.

    ## useButtonGroupCollection

    Retrieve the collection of buttons within a group.

    ### Syntax

    ```typescript
    function useButtonGroupCollection(): () => ItemData[];
    ```

    ### Parameters

    None.

    ### Return Value

    A function that returns an array (`ItemData[]`) representing the collection of items within the group. Each item has an `id` property of type `string`.

    ### Example

    ```typescript
    // Example of using the function
    const getItemCollection = useButtonGroupCollection();
    const collection = getItemCollection();
    console.log(collection);
    ```

    ### Remarks

    None.
