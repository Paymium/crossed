# Accordion
The `Accordion` component is used to show and hide sections of content within a limited space.
It allows users to expand or collapse information, helping them focus on relevant details without overwhelming the interface.

Accordions should be used to organize content hierarchically or group related information. Avoid nesting too many levels to maintain clarity and usability.

```tsx
import { Accordion } from "@crossed/ui"
```
