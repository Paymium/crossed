# Tabs
The `Tabs` component is used to organize and navigate between related sections of content within the same context.
It allows users to switch views without leaving the current page, keeping the experience fast and focused.

Tabs should be used to group content at the same hierarchy level. Avoid using too many tabs or nested tab structures to maintain clarity and ease of navigation.

```tsx
import { Tabs } from "@crossed/ui"
```
