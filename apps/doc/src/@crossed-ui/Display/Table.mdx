# Table

The `Card` component is used to group related information within a distinct container.
It helps organize content such as text, images, and actions into a visually separated block thatâ€™s easy to scan and understand.

Cards should be used to highlight key information or interactive elements. Avoid nesting multiple cards or overusing them, as it can clutter the interface and reduce visual hierarchy.
