# Badge
The `Badge` component is used to display small pieces of information or status indicators.
It can highlight counts, statuses, or categories to draw attention to important details without disrupting the layout.

Badges should be used sparingly to emphasize relevant information. Avoid overusing them, as too many badges can reduce their visual impact and clarity.

```tsx
import { Badge } from "@crossed/ui"
```
