# Group

The `Group` component is used to arrange multiple elements together in a horizontal or vertical layout.
It helps maintain consistent spacing and alignment between related components, such as buttons or inputs.

Groups should be used to create clear visual relationships between elements. Avoid grouping unrelated components to keep the interface organized and intuitive.

```tsx
import { Group } from "@crossed/ui"
```
