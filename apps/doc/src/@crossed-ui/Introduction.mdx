# Introduction

`@crossed/styled` is here to streamline your styling workflow, offering
the same ease of use as style-components but with a cross-platform twist.  
Additionally,
CSS simplifies maintenance and scalability, allowing for consistent style changes
and design expansion as the website evolves.

- <b>Unified Codebase:</b> Maintain a single codebase to generate styles
  seamlessly for both React and React Native platforms.
