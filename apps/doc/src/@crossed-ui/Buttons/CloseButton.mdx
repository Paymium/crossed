# CloseButton

The `CloseButton` component is used to dismiss dialogs, modals, or other temporary UI elements.
It represents a neutral action that allows users to exit or cancel a current interaction without triggering side effects.

Close buttons should be used consistently to provide a clear exit path. Avoid placing multiple close buttons in the same context to prevent confusion.

```jsx
import { CloseButton } from '@crossed/ui';
```
