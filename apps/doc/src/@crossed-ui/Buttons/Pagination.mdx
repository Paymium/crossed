# Pagination

The `Pagination` component is used to navigate through large sets of content divided into discrete pages.
It helps users move forward or backward efficiently, providing context about their current position within the content.

Pagination should be used when presenting content in chunks to avoid overwhelming users. Avoid displaying too many page links at once to maintain clarity.

```jsx title="Import Button"
import { Pagination } from '@crossed/ui';
```
