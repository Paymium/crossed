# Button

The `Button` component is used to trigger actions or events.
It can represent primary, secondary, tertiary, success, or error actions depending on the context.

Buttons should be used sparingly for key actions to help users achieve their goals efficiently.
Avoid using too many buttons with equal visual weight to prevent confusion.

```jsx title="Import Button"
import { Button } from '@crossed/ui';
```
