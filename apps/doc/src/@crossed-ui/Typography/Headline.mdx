# Headline
The `Headline` component is used to display prominent titles or headings within the interface.
It helps structure content hierarchically, guiding usersâ€™ attention and improving readability.

`Headline` should be used for key sections or important messages. Avoid using multiple headlines with equal visual weight in the same context, as it can reduce clarity and hierarchy.

```jsx
import { Headline } from '@crossed/ui';
```
