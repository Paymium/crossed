# Text
The `Text` component is used to display readable content in the interface.
It allows developers to apply typography styles, such as font size, weight, and color, consistently across the application.

`Text` should be used for all textual content to maintain visual consistency. Avoid using plain HTML or system text elements directly, as it can lead to inconsistent styling and accessibility issues.

```jsx
import { Text } from '@crossed/ui';
```
