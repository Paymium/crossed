# Sheet

The `Sheet` component is used to display content in an overlay panel that slides in from the edge of the screen.
It allows users to focus on secondary or contextual information without leaving the current view.

Sheets should be used for supplementary actions or content that doesnâ€™t require a full page. Avoid using multiple sheets simultaneously, as it can overwhelm users and complicate navigation.

```jsx
import { Sheet } from '@crossed/ui';
```
