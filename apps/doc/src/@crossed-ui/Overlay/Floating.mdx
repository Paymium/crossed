# Floating
The `Floating` component is a primitive used to anchor content to a trigger element.
It allows developers to display overlays, popovers, or tooltips relative to a reference element, ensuring proper positioning and visibility.

`Floating` should be used to create contextual UI elements that appear on interaction. Avoid overusing it for static content, as it is designed specifically for temporary, triggered displays.

```jsx
import { Floating } from '@crossed/ui';
```
