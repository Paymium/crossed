# Tooltip

The `Tooltip` component is used to provide additional information or context when users hover over or focus on an element.
It helps clarify functionality or content without cluttering the interface.

Tooltips should be used sparingly to support understanding of interface elements. Avoid placing essential information solely in tooltips, as it may be inaccessible on some devices or to assistive technologies.

```jsx
import { Tooltip } from '@crossed/ui';
```
