# Divider
The `FocusScope` component is a React container that manages the keyboard focus cycle within a specific section of the interface.
It allows you to trap focus inside a defined area (focus trapping) when an interactive element is open — such as a modal, popover, or menu — and restore it automatically once the element is closed.

```jsx
import { Divider } from '@crossed/ui';
```
