# VisibilityHidden
The `VisibilityHidden` component is used to hide content visually while keeping it accessible to screen readers.
It allows developers to improve accessibility without affecting layout or visual presentation.

`VisibilityHidden` should be used for content that needs to be read by assistive technologies but not displayed on screen. Avoid hiding interactive elements that users need to see or interact with, as it can create confusion.
