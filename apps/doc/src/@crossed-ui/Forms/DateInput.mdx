# DateInput
The `DateInput` component is a React input field that enables users to enter or select a date value.
It combines manuals text entry with optional calendar integration, supports locale-based formatting, and ensures consistent date handling for forms, filters, and scheduling interfaces.

```jsx
import { DateInput } from '@crossed/ui';
```
