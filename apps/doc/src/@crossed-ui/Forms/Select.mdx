# Select
The `Select` component is a React input control that allows users to choose one or multiple options from a dropdown list.
It supports searchable lists, custom rendering, and validation states, making it ideal for forms, filters, and dynamic selection interfaces.

```jsx
import { Select } from '@crossed/ui';
```
