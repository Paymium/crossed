# Calendar

The `Checkbox` component is a React input control that allows users to toggle between checked and unchecked states.
It is commonly used in forms and settings to enable or disable options, select multiple items from a list, or confirm user consent through a simple and accessible interface.

```jsx
import { Checkbox } from '@crossed/ui';
```
