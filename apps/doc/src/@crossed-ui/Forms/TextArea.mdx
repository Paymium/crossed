# TextArea
The `TextArea` component is a React input field that allows users to enter and edit multi-line text.
It supports adjustable sizing, validation states, and helper texts, making it ideal for forms, comments, notes, and any interface requiring longer text input.

```jsx
import { TextArea } from '@crossed/ui';
```
