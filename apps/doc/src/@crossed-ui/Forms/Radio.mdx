# Radio

The `Radio` component is a React input control that allows users to select a single option from a group of choices.
It ensures mutually exclusive selection, supports labels and validation states, and is commonly used in forms, surveys, and preference settings.

```jsx
import { Radio } from '@crossed/ui';
```
