# Banner

The `Banner` component is used to display important messages or alerts to users.
It can communicate information such as warnings, updates, or confirmations in a prominent yet non-intrusive way.

Banners should be used for high-priority messages that require user attention. Avoid overusing them, as too many banners can overwhelm the interface and reduce their effectiveness.

```jsx
import { Banner } from '@crossed/ui';
```
