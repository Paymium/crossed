# Usage

`@crossed/styled` allows you to create reusable styles for React Native components and apply them in multiple ways.

---

## Type information

Both `createStyle` and `inlineStyle` return a `CrossedMethods` object. This object provides access to:

* `.rnw()` for React Native styles
* `.className()` for web class names
* `.style()` for raw style objects

This allows you to write cross-platform components easily.

---

## Create multiple styles

You can define reusable styles using `createStyle`. Each key defines a style, and the `base` property contains the style values.

```tsx
import { createStyle } from '@crossed/styled';

export const styles = createStyle(() => ({
  title: {
    base: {
      fontWeight: '800',
      fontSize: '32px',
      textDecoration: 'underline',
    },
  },
  center: {
    base: {
      display: 'flex',
      flexDirection: 'column',
      justifyContent: 'center',
      alignItems: 'center',
    },
  },
  button: {
    base: {
      padding: 12,
      borderRadius: 8,
      backgroundColor: '#007bff',
    },
    ':hover': {
      backgroundColor: '#0056b3',
    },
    ':active': {
      backgroundColor: '#00408d',
    },
    ':focus': {
      borderColor: '#ff9900',
      borderWidth: 2,
    },
  },
}));
```

> This example shows how to define `hover`, `active`, and `focus` states in addition to the `base` style.

---

## Apply styles in a component

You can apply the created styles to your components using the `.rnw()` method.

```tsx
import { Text, View, type TextProps } from 'react-native';
import { type CrossedMethods } from '@crossed/styled';
import { styles } from './style';

type MyComponentProps = Omit<TextProps, 'style'> & {
  style?: CrossedMethods<any>;
  title: string;
};

export const MyComponent = ({ title, style, ...props }: MyComponentProps) => {
  return (
    <View {...styles.center.rnw()}>
      <Text {...props} {...styles.title.rnw()}>
        {title}
      </Text>
    </View>
  );
};
```

> `.rnw()` applies the style in a React Native compatible way. Here, the `View` is centered and the text uses the `title` style.

---

## Hover, Active, and Focus states

When working with interactive components such as buttons, you can define styles for `hover`, `active`, and `focus` states. Use `rnw()` with an object containing `{ hover, active, focus }` to apply these states dynamically.

```tsx
import { Pressable, Text, type TextProps } from 'react-native';
import { composeStyles, type CrossedMethods } from '@crossed/styled';
import { styles } from './style';

type ButtonProps = Omit<TextProps, 'style'> & {
  style?: CrossedMethods<any>;
  title: string;
  disabled?: boolean;
};

export const Button = ({ title, style, disabled, ...props }: ButtonProps) => {
  return (
    <Pressable
      style={({ pressed, hovered, focused }) =>
        composeStyles(styles.button, disabled && styles.disabled, style).rnw({
          active: pressed,
          hover: hovered,
          focus: focused,
        }).style
      }
    >
      <Text {...props} {...styles.title.rnw()}>
        {title}
      </Text>
    </Pressable>
  );
};
```

> This allows the button to respond visually to hover, active (pressed), and focus states in a cross-platform way.
